# CYberAware 🔐

A structured cybersecurity awareness and attack simulation platform designed to demonstrate modern cyber threat classification, attack lifecycle understanding, and defensive security strategies.

---

## 📌 Project Overview

CYberAware is a portfolio-driven cybersecurity platform that explains major attack categories in a structured, technical, and practical manner.

The project includes:

- Attack classification aligned with the **CIA triad**
- References to the **MITRE ATT&CK framework**
- Interactive attack simulations (safe, frontend-only)
- Detection indicators and mitigation strategies
- Structured learning resources

This project was developed to bridge theoretical cybersecurity knowledge with practical attack modeling.

---

## 🧠 Key Features

### 1️⃣ Attack Modules
Structured breakdown of:

- Phishing & Social Engineering  
- Malware  
- Denial of Service (DoS)  
- Man-in-the-Middle (MitM)  
- Injection Attacks  
- Password Attacks  
- Insider Threats  
- Zero-Day Exploits  
- Advanced Persistent Threats (APT)  
- Supply Chain Attacks  
- IoT & Hardware Attacks  
- Cryptographic Attacks  

Each module includes:

- Attack mechanism explanation
- CIA impact analysis
- Detection indicators
- Defensive strategies
- MITRE ATT&CK mapping

---

### 2️⃣ Interactive Simulations

Frontend-only safe simulations for:

- Phishing credential capture demonstration
- Malware behavior simulation
- Man-in-the-Middle traffic interception
- Password strength & brute-force visualization
- APT kill chain progression

⚠ No real attack execution. No data collection. No backend storage.

---

### 3️⃣ Structured Threat Modeling

The platform demonstrates understanding of:

- Attack lifecycle stages
- Lateral movement concepts
- Data exfiltration techniques
- Secure coding awareness
- Threat detection methodology
